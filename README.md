# Student Location Tracking System

## Overview

The Student Location Tracking System is an Android-based application designed to provide real-time location tracking of students and staff within a college campus. The system enhances campus safety, improves attendance monitoring, and enables efficient campus management through GPS-based location services and digital campus mapping.

## Features

* Real-time student location tracking
* Staff and student login modules
* GPS-based location monitoring
* Campus map integration
* Attendance monitoring support
* Emergency response assistance
* Location-based notifications and alerts
* Firebase integration for real-time updates
* MySQL database management
* User-friendly Android interface

## Technologies Used

* Java
* Android Studio
* Firebase
* MySQL
* XAMPP
* Google Location Services
* REST APIs
